# React + Tailwind Week 3 

## Features
- Vite + React
- Tailwind CSS (dark mode via `class`)
- React Router (Home, Tasks, Posts)
- Task Manager with `useState`, `useEffect`, and `useLocalStorage` hook
- API integration: JSONPlaceholder posts with pagination and search
- Reusable components: Button, Card, Navbar, Footer, Layout
- Theme switcher (light/dark)

## Setup
1. Node.js v18+ recommended
2. Install deps:
```bash
npm install
```
3. Start dev server:
```bash
npm run dev
```

## Notes
- Tailwind and PostCSS are configured (`tailwind.config.js`, `postcss.config.js`).
